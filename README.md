# Smart Sales Monitoring & Optimization System

An interactive dashboard that explores how social media, economic cycles,
calendar effects, and marketing interventions drive daily sales for an SME
in Sulaymaniyah, Kurdistan Region of Iraq.

**Authors:** Paywand Yousif Aziz, Amanj Salih Mahmud, Shnyar Baram Mohamad
**Supervisor:** Dr. Miran Taha
**University of Sulaimani — Department of Computer Science, 2026**

---

## Sections

- **About** — project overview
- **Overview** — KPI cards, monthly product performance, weekly rhythm
- **Findings** — 8 interactive analyses derived from 361 days of indexed sales data
- **Conclusion** — summary, limitations, and future directions

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

The app will open at <http://localhost:8501>.

## Deploy

Deployed on Streamlit Community Cloud. Any push to `main` auto-redeploys.
